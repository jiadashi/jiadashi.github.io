1.字符串的声明：

var str="我是字符串" //推荐

str="我是字符串"

注意：一个空格算一个长度。汉字算一个长度。 连续两个空格算两个。相当于光标移动的次数

常用方法

1.indexof()  检索字符串，返回指定字符串的值在字符串首次出现的位置。若没有，则返回-1.
2.match() ,在字符串内检索指定的值，找到或多个正则表达式的匹配（好绕口，不太懂）。
例如： var str="hello Mr Mi";
    document.write(str.match('Mi'))    输出为：Mi  
    若未匹配到，返回值为“null”；
3.substr (start,length);截取字符串，两个参数表示从哪截，截多长。start值必设，length不设的话，默认截到最后，length若为负值，则截取为空。
4.substring(from,to);截取字符串，表示从哪截，截到哪。to不设，默认截到最后。

5.split(); 把一个字符串分割成字符串数组；
例如：var str = "[ti:The Fox][00:00.00] The Fox - Ylvis";
        
         var str2=str.split("]");
         
         document.write(str2+"
");
         lyric = str2[2];
         document.write("歌词是："+lyric+"
");
         var timea = str2[1];
         var timeb = timea.split("[");
         document.write("时间是："+timeb[1]+"
");
         var namea = str2[0].split("[");
         var nameb = namea[1].split(":");
         document.write("歌名是："+nameb[1]);

数组的常见的操作方法

1.push();   方法可向数组的末尾添加一个或多个元素，并返回新的长度。
说明：push() 方法可把它的参数顺序添加到 arrayObject 的尾部。它直接修改 arrayObject，而不是创建一个新的数组。push() 方法和 pop() 方法使用数组提供的先进后出栈的功能。  
例如：   
var arr = ["George","John","Thomas"]
document.write(arr + "
")
arr.push("James") 
document.write(arr)
输出为：George,John,Thomas
George,John,Thomas,James

2.unshift();向数组中添加元素，位置为第一个。直接操作数组。
3.shift(); 从数组删除元素，位置为最后一个。直接操作数组。
4.slice(start,end)   返回一个新的数组，从start（包括）到end（不包括） ；	请注意，
该方法并不会修改数组，而是返回一个子数组。如果想删除数组中的一段元素，应该使用方法 Array.splice()。
